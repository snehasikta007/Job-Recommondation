AI-powered web application for personalized job and career recommendations.
Analyzes user resumes to extract skills using AI/NLP techniques.
Matches user skills with job requirements and calculates job match percentage.
Identifies skill gaps and recommends skills to improve employability.
Supports both freshers and experienced professionals.
Secure user registration, login, and profile management.
Responsive and user-friendly interface inspired by modern job portals.
Built using HTML, CSS, JavaScript (Frontend), Python (Flask) (Backend), and SQLite (Database).
